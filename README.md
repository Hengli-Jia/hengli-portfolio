# Hengli Jia Portfolio Website

A minimalist bilingual portfolio/resume website for Hengli Jia.

## Files

- `index.html`
- `style.css`
- `script.js`

## How to deploy with GitHub Pages

1. Create a GitHub repository, for example `hengli-portfolio`.
2. Upload these three files to the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. GitHub will generate a free public website URL.
