## Deploy on GitHub Pages

1. Upload `index.html`, `style.css`, and `script.js` to your GitHub repository.
2. Go to **Settings → Pages**.
3. Choose **Deploy from a branch**.
4. Select `main` and `/root`.
5. Save.
